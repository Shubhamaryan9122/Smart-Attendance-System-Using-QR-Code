# QR Attendance System

A QR Code Based Attendance Management System developed using PHP, MySQL, HTML, CSS and JavaScript.

## Features

- Teacher Login
- Student Login
- Dynamic QR Generation
- QR Based Attendance
- Secure Token Validation
- Auto Refresh QR
- Attendance Tracking
- Add Students

## Technologies Used

- PHP
- MySQL
- JavaScript
- HTML/CSS
- XAMPP
